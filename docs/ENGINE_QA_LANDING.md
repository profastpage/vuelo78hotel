# Landing QA

## Core Objective

Landing pages must feel conversion-first, fast to scan, and focused on one dominant action.

## Required Checks

- hero is readable in one glance
- main CTA is obvious
- proof/testimonials feel credible
- FAQs reduce friction
- final section closes with a clear action
- typography remains balanced at `320`, `375`, `768`, `1440`
- no decorative layout decision weakens conversion flow

## Failure Examples

- hero too artistic and unclear
- CTA below the fold without reason
- weak hierarchy between message and proof
- empty visual blocks
- mobile layout that feels collapsed or accidental
