# Style Archetypes

These style definitions are treated as composition systems, not skin swaps.

## Core Rules

- A style change must alter hero composition, grid logic, section rhythm, hierarchy, card behavior, and responsive collapse.
- No style may be reduced to typography, radius, shadow, or color-only changes.
- Mobile must preserve the style direction without keeping fragile desktop overlap.

## Reference Basis

This catalog was informed by current interface guidance and contemporary web patterns:

- Apple Liquid Glass and Human Interface principles: depth, material hierarchy, motion restraint, and readability.
- Material-style adaptive layout principles: compositional changes must remain stable across breakpoints.
- Current web patterns around product-led SaaS, bento grids, editorial asymmetry, immersive media, and motion-first UI.

## Style Notes

- `Normal`: balanced default, stable split hero, moderate rhythm, commercially safe.
- `Minimalista`: clean split, large spacing, low density, few large cards, editorial calm.
- `Glassmorphism`: floating translucent panels, controlled overlap, blur and glow, layered depth.
- `Bento`: modular irregular grid, mixed card sizes, dashboard-like hierarchy.
- `Editorial`: asymmetric layout, dominant type, stronger narrative rhythm, offset media.
- `Immersive`: dominant visual hero, cinematic sections, large scenes, low block count.
- `SaaS Modern UI`: product-first hero, proof cards, workflow sections, conversion clarity.
- `Minimal Business`: executive and sober, trust-first, stable business grid.
- `Web visual con imagenes grandes`: image-led layout, editorial media emphasis, reduced copy density.
- `Interactive Motion UI`: movement-supported hierarchy, layered panels, motion-aware sections.
- `Luxury`: restrained premium pacing, few large modules, high-value whitespace.
- `Tecnologico`: system-driven panels, utility cards, technical density with order.
- `Espacial`: atmospheric spacing, glow fields, sparse asymmetry, futuristic calm.
- `Moderno`: current, fluid, polished, mixed rhythms and flexible modules.
- `Editor`: curated magazine-like composition, typographic rhythm, framed media.
- `Premium`: polished richness, showcase hero, layered proof and premium shells.
- `Pro`: operational clarity, efficient utility composition, low ornament.
- `3D`: depth-led scenes, layered modules, dimensional cues without UX damage.
- `9bit`: retro modularity, chunkier cards, stylized but readable structure.
